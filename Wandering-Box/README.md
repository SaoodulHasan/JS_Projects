# 🎯 Wandering Box

A fun reflex-based browser game built using **HTML**, **CSS**, and **Vanilla JavaScript**.

The objective is simple:
Hover over the box as it randomly moves around the screen and click it before it escapes. Miss a click anywhere inside the game area, and it's **Game Over!**


## ✨ Features

- 🎮 Interactive gameplay
- 📦 Random box movement
- 🖱️ Hover-based mechanics
- ⏱️ Countdown timer
- 📈 Live score tracking
- 💀 Game Over on missed click
- 🎨 Modern responsive UI
- 🔄 Restart functionality

## 🎯 Inspiration

The idea for this project came after seeing a similar JavaScript game shared on LinkedIn. Instead of following a tutorial or using the source code, I challenged myself to build it entirely from scratch as a way to revise my frontend fundamentals after almost a year.

I first focused on getting the game logic working and then polished the UI using design inspirations.

> **First make it work. Then make it beautiful.**
