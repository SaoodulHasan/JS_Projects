# 🎬 MoviBuzz

MoviBuzz is a modern movie discovery web application built using **HTML, CSS, and JavaScript**. It uses the **TMDB (The Movie Database) API** to fetch real-time movie information and provides an intuitive interface for exploring movies.

---

## ✨ Features

* 🔥 Browse **Trending Movies**
* ⭐ View **Popular Movies**
* 🏆 Explore **Top Rated Movies**
* 🎥 Discover **Upcoming Movies**
* 🔍 Search for movies by title
* 📄 Dedicated movie details page featuring:

  * Movie poster & backdrop
  * Overview
  * Rating
  * Runtime
  * Release date
  * Budget
  * Genres
  * Director
  * Writers
  * Cast
  * Similar movie recommendations
* 🌙 Dark Mode UI
* 📱 Responsive design

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/SaoodulHasan/MoviBuzz.git
```

### 2. Open the project

Simply open `index.html` in your browser, or use a local development server such as **Live Server** in VS Code.

---

## 🔑 TMDB API Setup

This project uses the **TMDB API**.

1. Create an account on TMDB.
2. Generate an **API Read Access Token**.
3. Replace the placeholder token in your JavaScript file:

```javascript
Authorization: "Bearer [ACCESS TOKEN]"
```
